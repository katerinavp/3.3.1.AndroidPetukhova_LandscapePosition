# 3.3.1.Calculator_LandscapePosition
Разработана панель действий калькулятора.  
- В качестве основного Layout экрана используется RelativeLayout.  
- Реализованы действия калькулятора:  
сложение  
умножение  
вычитание  
деление  
процент  
получение результата (=)  
очистка поля (С)  
смена знака  (+/-)
